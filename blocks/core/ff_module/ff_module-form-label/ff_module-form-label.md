---
data:
    text: "Stacked text input name"
    forId: "label-id"
    required: false
    data: 
        -
            attr: data-ff-label
            value: "{'variables': 15}"
---

## Modifier options:

- heading
- required

Data-attributes are set by attr:value pairs, eg
```
    id: "input-id"
    value: "Form input"
    name: "input-name"
    data:
        -
            attr: data-ff-attr
            value: true
```
